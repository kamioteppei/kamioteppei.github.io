# React Native

## 基本

- iOS用と、Andoroid用のアプリを、Reactでコードを共有して開発できる。  
- 100%共有できる訳ではない。お互いのNativeの呼び出しであったり、  
osに依存する部分は個別の実装になる。

## コンポーネント

- htmlやcssは使用しない。
- [コンポーネント](https://nativebase.io)を使用する。
