# jupyter notebook

## インストール
アナコンダをインストール

## ローカルサーバーを起動する
```
$ cd <jupyter notebook>
$ jupyter notebook
```

## ローカルサーバーをシャットダウンする
```
$ (crtl + c)
```

## カーネル操作

セルで、bashを実行するには、コマンドの先頭に`!`を付ける
```
!pip install -q keras
```
からの
```
import keras
```
